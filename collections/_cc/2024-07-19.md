# 每周一道算法题：
274. H 指数 
```java
class Solution {
    public int hIndex(int[] citations) {
        Arrays.sort(citations); // 先对引用次数数组进行排序
        int hIndex = 0;
        for (int i = citations.length - 1; i >= 0; i--) {
            int rank = citations.length - i; 
            if (citations[i] >= rank) { 
                hIndex = rank; 
            } else {
                break; 
            }
        }
        return hIndex;
    }
}
```
