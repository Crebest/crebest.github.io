122.买卖股票的最佳时机II：
```java
class Solution {
    public int maxProfit(int[] prices) {
        int minFlag = Integer.MAX_VALUE;
        int sum = 0;
        for(int i = 0;i < prices.length;i++){
            if(prices[i] > minFlag) {
                int temp = prices[i] - minFlag;
                sum += temp;
            }
            minFlag = prices[i];
        }
        return sum;
    }
}
```
