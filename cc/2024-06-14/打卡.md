# 每周一道算法题：
189.轮转数组：
```java
class Solution {
    public void rotate(int[] nums, int k) {

        k = k % nums.length;

        int[] res = new int[nums.length];

        for(int i = 0;i < k;i++) {
           res[i] = nums[nums.length - k + i];
        }

        for(int i = k;i < nums.length;i++) {
            res[i] = nums[i - k];
        }

        for(int i = 0;i < nums.length;i++) {
            nums[i] = res[i];
        }
    }
}
```
